How It Works

This site uses the primitive-haxe library to recreate images as geometric primitives. Given a target image to recreate with shapes, the library generates random shapes and repeatedly mutates them as part of a hillclimbing optimization approach, only picking the best-fitting shapes. The settings can be changed interactively via the settings dropdown to find a good balance of speed and quality.
